# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability, please do not open a public issue.

Please send an email directly to: yanhuiwu7@163.com

Include the following:
- Description of the vulnerability
- Steps to reproduce
- Impact scope
- Possible solution (if any)

We will evaluate and fix the issue as soon as possible.

## Supported Versions

We provide security patches for the latest version. Please ensure you are using the latest plugin version.

## Known Security Limitations

- Plugin loads D3.js via CDN (https://d3js.org/d3.v7.min.js), requires network connection for first load
- Plugin does not collect or transmit any user data
- All graph data is stored locally in Obsidian Vault
